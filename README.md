# Air Quality Index (AQI) Analysis Dashboard

## Project Overview
This project analyzes air pollution data across India using Microsoft Power BI. The dashboard provides insights into pollution patterns, dominant pollutants, and air quality distribution using interactive visualizations.

## Tools Used
- Microsoft Power BI
- Power Query
- Data Visualization

## Dashboard Features
- Average AQI indicator
- Pollution distribution map of India
- Pollutant comparison chart
- Dominant pollutant pie chart
- Interactive pollutant filter

## Key Insights
- PM2.5 and PM10 contribute significantly to air pollution levels.
- Pollution hotspots can be identified using geographic visualization.
- Interactive filters allow dynamic exploration of pollution data.
